여러 이미지 합치는 프로그램 만들기

[사용자]
사용자는 합치려는 이미지 1개 이상 선택
합쳐진 이미지가 저장될 경로 지정
가로넓이, 간격, 포맷 옵션을 지정한다. 
시작 버튼을 통해 이미지를 합친다.
닫기 버튼을 통해 프로그램을 종료한다. 

[기능 명세]
파일 추가 : 리스트 박스에 파일 추가
선책 삭제 : 리스트 박스에서 선택된 항목 삭제
찾아보기 : 저장 폴더를 선택하면 텍스트 위젯에 입력
가로넓이 : 이미지 넓이 지정 (원본 유지, 1024, 800, 640)
간격 : 이미지 간의 간격 지정 (없음, 좁게, 보통, 넓게)
포맷 : 저장 이미지 포맷 지정 (png, jpg, bmp)
시작 : 이미지 합치기 작업 실행
진행상황 : 현재 진행중인 파일 순서에 맞게 반영
닫기 : 프로그램 종료
